## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## ORM Config

Create orm.config.ts and use the value here:

```
import { TypeOrmModuleOptions } from '@nestjs/typeorm';
import { join } from 'path';

export const configuration: TypeOrmModuleOptions = {
  type: 'postgres',
  host: 'YOUR HOST',
  port: 5432,
  username: 'USERNAME',
  password: 'PASSWORD',
  database: 'DATABASE',

  entities: [join(__dirname, './**/*.entity{.ts,.js}')],
};

```
